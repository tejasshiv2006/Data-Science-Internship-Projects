# Titanic Survival Prediction Using Machine Learning

## Project Overview

This project uses machine learning techniques to predict whether a passenger survived the Titanic disaster. The dataset contains passenger information such as age, gender, ticket class, fare, and family details.

## Objectives

* Perform data cleaning and preprocessing.
* Train machine learning models for prediction.
* Compare model performance.
* Evaluate results using Confusion Matrix and ROC Curve.

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-learn

## Machine Learning Models

1. Decision Tree Classifier
2. Random Forest Classifier

## Workflow

1. Data Loading
2. Data Cleaning
3. Feature Engineering
4. Train-Test Split
5. Model Training
6. Model Evaluation
7. Performance Visualization

## Evaluation Metrics

* Accuracy Score
* Precision
* Recall
* F1-Score
* Confusion Matrix
* ROC Curve

## Conclusion

The Random Forest model achieved better performance than the Decision Tree model and was selected as the final predictive model for Titanic survival prediction.

## Learning Outcomes

* Understanding supervised learning.
* Applying classification algorithms.
* Evaluating model performance.
* Visualizing results using Confusion Matrix and ROC Curve.
