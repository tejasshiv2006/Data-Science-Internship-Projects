# Retail Sales Analysis Project

## Project Overview

This project performs exploratory data analysis (EDA) and sales prediction on the Superstore retail dataset. The goal is to identify business insights, understand sales patterns, and apply machine learning techniques to predict sales.

---

## Dataset

**Dataset Used:** Sample - Superstore.csv

The dataset contains information about:

* Orders
* Sales
* Profit
* Discount
* Quantity
* Category
* Sub-Category
* Region
* State

---

## Objectives

* Clean and preprocess data
* Perform exploratory data analysis (EDA)
* Visualize sales and profit trends
* Identify top-performing categories and regions
* Analyze correlations among variables
* Build a machine learning model for sales prediction

---

## Tools and Libraries

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-Learn

---

## Analysis Performed

### Data Cleaning

* Checked missing values
* Checked duplicate records

### Data Visualization

* Sales by Category
* Profit by Category
* Sales by Region
* Top 10 States by Sales
* Sales Distribution Histogram
* Correlation Heatmap

### Machine Learning

* Linear Regression Model
* Sales Prediction
* Model Evaluation using:

  * MAE
  * RMSE
  * R² Score

---

## Key Insights

* Technology category generated high sales.
* Sales varied significantly across regions.
* Some states contributed more revenue than others.
* Discount and Profit showed relationships with Sales.
* Linear Regression was used to predict future sales.
